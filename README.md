## 🚀 Истифода

Барои кушодан кифоя аст файлро дар браузер кушоед:

## 🌐 Deploy кардан

### Netlify Drop (осонтарин)

2. Файли HTML-ро drag & drop кунед
3. Линки доимӣ мегиред ✅

### GitHub Pages
1. Repository нав созед
2. Файлро upload кунед
3. Settings → Pages → Deploy кунед ✅

1. [vercel.com](https://vercel.com) кушоед
2. Файлро import кунед ✅
